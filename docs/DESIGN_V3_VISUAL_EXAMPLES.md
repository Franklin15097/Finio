## 🎨 Компоненты

### 1. AnimatedCounter

```tsx
// Простой счетчик
<AnimatedCounter value={125430} suffix=" ₽" />

// С анимацией
125,430 ₽
  ↓ (плавно меняется)
130,250 ₽
```

**Эффект:** Цифры плавно "перетекают" от старого значения к новому

## 💡 Советы по использованию

### 1. Комбинируйте эффекты
```tsx
<GlassCard 
  className="hover-lift animate-fade-in"
  hover
>
  <AnimatedCounter value={1000} />
</GlassCard>
```

### 2. Используйте stagger для списков
```tsx
{items.map((item, i) => (
  <div className={`animate-slide-up stagger-${i + 1}`}>
    {item}
  </div>
))}
```

### 3. Добавляйте интерактивность
```tsx
<AnimatedDonutChart
  data={data}
  onSegmentClick={(segment) => {
    // Показать детали
  }}
/>
```
