### CircularProgress

```
        ╭─────────╮
       ╱           ╲
      │             │
      │    75%      │  ← Процент
      │   Еда       │  ← Подпись
      │             │
       ╲           ╱
        ╰─────────╯
         ▓▓▓▓▓░░░    ← Прогресс
```

**Цвета:**
- 0-69%: Зеленый (успех)
- 70-89%: Желтый (предупреждение)
- 90-100%: Красный (превышение)

**Анимация:**
- Заполнение: 1.5 сек
- Easing: cubic-bezier (ease-out)
- Glow эффект

### PullToRefresh

```
     ↓ Потяните для обновления
┌─────────────────────────────────┐
│         🔄                      │
│                                 │
│  [Контент]                      │
│                                 │
└─────────────────────────────────┘
```

**Состояния:**
1. Idle: Нет индикатора
2. Pulling: Иконка появляется
3. Ready: Иконка полностью видна
4. Refreshing: Вращение
5. Success: Checkmark

## 🎬 Анимации

### Появление элементов

```
1. fadeIn
   ↓ Снизу вверх + fade
   
2. fadeInScale
   ○ → ● Масштабирование + fade
   
3. slideInRight
   → Справа налево
   
4. slideInLeft
   ← Слева направо
```

### Циклические анимации

```
1. bounce
   ↑↓ Подпрыгивание
   
2. pulse
   ○ ◉ ○ Пульсация
   
3. shimmer
   ▓▓▓░░░ Мерцание
   
4. glow
   ○ ◉ ○ Свечение
   
5. float
   ↑↓ Плавание
   
6. rotate
   ↻ Вращение
```

### Hover эффекты

```
1. hover-lift
   ↑ Подъем на 4px
   
2. hover-scale
   ○ → ● Увеличение 1.05x
   
3. hover-glow
   ○ → ◉ Свечение
```
